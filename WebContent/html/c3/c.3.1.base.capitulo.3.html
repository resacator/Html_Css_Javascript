border-image

 Permite asociar una imagen como elemento generador del borde definido.
El ultimo parametro admite los valores:
- repeat: el aspecto de la imagen no se altera y se recorta si es necesario
- round: determina la longitud del largo a cubrir y ajusta el tamaño de las piezas para redondear a las N piezas enteras necesarias
- stretch: usa una sola imagen y la expande por la totalidad del area del borde.

border-radius

 Permite redondear las esquinas de una figura.
Admite hasta 4 valores:
4 valores separados por ',': supIz, supDe, InfDe, InfIz
2 valores separados por '/': horiz / vert
1 valor: para cada esquina usa el mismo valor

box-shadow

 Crea efecto sombra a un elemento.
El primer valor es el color (usando rgb, #XXXXXX...).
El segundo valor es la distancia horizontal.
El tercer valor es la distancia vertical.
El cuarto valor es opcional e indica la distancia de difuminado (Ojo porque AMPLIA la superficie que cubre...)
El quinto valor es opcional: 'inset', aplica el efecto al interior de la caja

linear-gradient

 propiedad linear-gradient(a,b X,c Y):
Permite realizar un degradado de color.
La propiedad se asocia al fondo (background).
el primer parametro determina la posición de inicio.
Se indica en pixeles, % o clave (top, right...).
El segundo parametro determina el color inicial. Admite punto de terminación del color.
el tercer parametro determina el color final. Admite punto de terminación del color.

radial-gradient

 Permite realizar un degradado de color. La propiedad se asocia al fondo (background).
El primer parametro determina la posición de inicio. Se indica en pixeles, % o clave (top, right...)
El segundo parametro determina la forma. Solo admite los valores circle y ellipse.
El tercer parametro determina el color inicial. Admite punto de terminación del color.
El cuarto parametro determina el color final. Admite punto de terminación del color.

rgba

 Funcion para asignar color. Es una versión mejorada de rgb().
El primer atributo es el valor rojo.
El segundo atributo es el valor verde.
El tercer atributo es el valor azul.
El cuarto atributo es el rango de opacidad, de 0 a 1.
Top

hsla

 Función para asignar color. Es una versión mejorada de hsl().
El primer atributo es el valor del tono: Representa el color extraído de una rueda imaginaria en la que:
- los grados cercanos a 0/360 contienen los colores rojos
- los grados cercanos a 120 contienen los colores verdes
- los grados cercanos a 240 contienen los colores azules.
El segundo atributo es el valor de la saturacion.
El tercer atributo es el valor.
El cuarto atributo es el rango de opacidad, de 0 a 1.

transform: scale

 transform: scale: permite presentar el elemento a una escala concreta, tomando como base su tamaño original
- los valores de 0 a 1 reducen el tamaño
- los valore superiores a 1 aumentan el tamaño
Admite dos valores, aplicados a la escala X y a la escala Y. Si se indica un solo valor se aplica a ambos

transform: skew

 transform: skew permite "inclinar" sobre los planos (horizontal, vertical) un elemento
- Los valores debe indicarse seguido de la palabra clave "deg"
- Cada valor es independiente, y afecta a una y solo una dimensión (X,Y), siendo ambos opcionales.
